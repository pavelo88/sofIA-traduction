# **App Name**: SoftIA Translate

## Core Features:

- Spatial AR Vision Lens: Utilizes on-device processing and Gemini as a cloud fallback to detect and translate text in real-time through the lens of a live camera interface.
- SoftIA Kitten Tutor: An interactive AI agent that acts as a tool to evaluate translation quality, guide reading, and simulate voice-based conversations for learning.
- Test Simulation Flow: Allows developers to bypass camera hardware constraints by uploading static images to test OCR and AI translation pipelines instantly.
- Bento Grid Dashboard: A Glassmorphism 2.0 based spatial UI to visualize learning progress metrics with high-performance 3D parallax effects.
- Thermal Throttling Manager: Monitors device simulation health to intelligently downscale resolution and FPS during heavy local AI processing tasks.
- Supabase Integration Hub: Secure storage for user vocabulary progress, conversational history, and app settings managed via Supabase database.

## Style Guidelines:

- Primary Color: Electric Violet (#A162F7) to convey high-tech digital innovation and creative spatial interfaces.
- Background: Deep Obsidian Purple (#141118) providing a dark, high-contrast canvas for vibrant translucent elements.
- Accent: Periwinkle Frost (#9BA8F5) to represent the softer assistant elements and emphasize secondary call-to-actions.
- Font pairing: 'Space Grotesk' (sans-serif) for high-impact tech-forward headlines and 'Inter' (sans-serif) for readability in body text and AI chats.
- Bento Grid architectural layout using deep spatial z-index layers and frosted-glass transparency effects.
- Tactile physics including 'squish effects' and hyper-realistic micro-interactions triggered by 3D tilt movement using Reanimated.
- Thin-weight line icons with subtle glowing halos to align with the Glassmorphism 2.0 aesthetic.
